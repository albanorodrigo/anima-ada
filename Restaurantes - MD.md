# Restaurantes - MD
Se tiene la siguiente información:  
<ul>
  <li>Las personas frecuentan algún restaurante. </li>
  <li>A las personas les gustan distintas comidas. </li>
  <li>Los restaurantes sirven comidas. </li>
</ul>

Construir el Modelo de Dominio y presentarlo en un diagrama utilizando UML, teniendo en cuenta las siguientes restricciones:  

<ul>
  <li>Un restaurante no sirve más de 10 comidas.  </li>
  <li>Una persona frecuenta varios restaurantes. </li>
  <li>A una persona no le gusta una comida por sí sola sino cómo la sirven en determinados restaurantes, aunque puede no gustarle ninguna. </li>
  <li>Una comida servida por un restaurante puede no gustarle a ninguna persona.  </li>
</ul>
